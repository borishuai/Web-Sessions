New Selectors

Finding elements by class (DOM API)
var el = document.getElementById('section1');
el.focus();

var els = document.getElementsByTagName('div');
els[0].focus();

var els = document.document.getElementsByClassName('section');
els[0].focus();
Finding elements by CSS syntax (Selectors API)
var els = document.document.querySelectorAll("ul li:nth-child(odd)");
var tds = document.document.querySelectorAll("table.test > tr > td");
var el = document.document.querySelector("table.test > tr > td"); // el == tds[0]

Custom data-* attributes

Define, store, and retrieve custom data on the DOM.
<div id="out" data-id="good" data-name="joe" data-screen-name="user1"></div>
// Add new data attributes via JS.
var el = document.querySelector('#out');
el.setAttribute('data-foo', 'bar');

var html = [];
for (var key in el.dataset) {
  html.push(key, ': ', el.dataset[key], '<br>');
}

el.innerHTML = html.join('');
Output
id: good
name: joe
screenName: user1
foo: bar
